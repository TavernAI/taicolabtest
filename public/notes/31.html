Kobold Horde

Kobold Horde is a computation distribution platform. That means it generates responses for you, and allows you to generate for others, anonymously.

You can register at https://horde.koboldai.net/register. If you don't want to, simply leave the API key empty. As a guest, your requests will have the lowest possible priority.

If you register and generate for others, you earn kudos, which help your own requests to be prioritized. For instance, you can run a lighter model, earn kudos, and use them to be prioritized when generating with more complex models.

If you already run KoboldAI and want to generate for others, simply follow these steps (for Windows):

  1. If you haven't already, register at https://horde.koboldai.net/register and claim your Horde API key
  2. Download KoboldAI-Horde-Bridge from https://github.com/db0/KoboldAI-Horde-Bridge
  3. Rename clientData_template.py to clientData.py
  4. Edit clientData.py. Change the kai_name, and input your Horde API key into api_key
  5. Launch your KoboldAI client and load up a model
  6. Launch bridge_start.bat. In the command prompt, you should see a connection being established to your KoboldAI, and requests being handled.

Your Horde Bridge should automatically prioritize requests made by your own API key.

If you are a part of Kobold Horde, thank you!

For more details, see: https://horde.koboldai.net.
