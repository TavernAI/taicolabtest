OpenAI API key

How to get:

1. Login to OpenAI through the website.

2. Сlick on your profile on the right-top.

3. Сhoose "View API Key" in menu.

4. Enter your api key in the api key pole.

How to use proxy:

1. Enter the proxy adress instead default OpenAI api url.

2. Enter the password for the proxy if required.

To restore the default OpenAI api url address, click the revert button.
